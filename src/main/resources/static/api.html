Overview

VulScan is an internal vulnerability management backend that correlates asset inventory data with public vulnerability sources. The API is consumed by a lightweight dashboard UI and inventory clients.

  • JWT authentication with access + refresh tokens
  • Inventory is imported as CSV from client scripts
  • Feed management for KEV, EPSS, and NVD
  • Risk views combine inventory, KEV, EPSS and NVD advisory signals
  • Assets search API supports host/software/CVE/risk filtering and suggestions

Authentication

POST /api/auth/login

Returns JWT access and refresh token pair.

{
  "email": "superadmin@vulscan.local",
  "password": "SuperAdmin123!"
}

POST /api/auth/refresh

Refreshes access token using refresh token.

{
  "refreshToken": "..."
}

User profile

GET /api/users/me

Authenticated user profile (USER/ADMIN/SUPERADMIN).

{
  "id": 1,
  "email": "superadmin@vulscan.local",
  "status": "ACTIVE",
  "roles": ["USER", "ADMIN", "SUPERADMIN"]
}

Admin users

GET /api/admin/users

List users (ADMIN/SUPERADMIN).

POST /api/admin/users

Create user (ADMIN/SUPERADMIN).

{
  "email": "user@company.local",
  "password": "ChangeMe123!",
  "status": "ACTIVE",
  "roles": ["USER"]
}

PUT /api/admin/users/{id}/roles

Update user roles (ADMIN/SUPERADMIN).

PUT /api/admin/users/{id}/status

Update user status (ADMIN/SUPERADMIN).

Dashboard

GET /api/dashboard/summary

Returns aggregated dashboard statistics.

{
  "inventoryRows": 404,
  "hostCount": 12,
  "kevCount": 1234,
  "lastInventorySeenAt": "2026-03-03T11:22:33"
}

GET /api/dashboard/hosts/recent?limit=5

Returns recent hosts list (limit 1..20).

[
  {
    "hostname": "PC-001",
    "itemCount": 42,
    "lastSeenAt": "2026-03-03T11:22:33"
  }
]

GET /api/dashboard/kev/latest

Latest KEV aggregate information for dashboard widgets.

Inventory

GET /api/inventory/health

Health check endpoint (USER/ADMIN/SUPERADMIN).

POST /api/inventory/import

Imports inventory items from CSV (multipart/form-data, field name: file).

{
  "rowsRead": 404,
  "rowsInserted": 404,
  "rowsRejected": 0,
  "errors": []
}

CSV format

hostname,product,vendor,version,installedOn,source,cpe

KEV Catalog

GET /api/kev/status

Returns last KEV refresh status.

{
  "refreshedAt": "2026-03-03T11:22:33+01:00",
  "vulnsRead": 1234,
  "upserted": 1234,
  "kevCountBefore": 1200,
  "kevCountAfter": 1234,
  "added": 34
}

POST /api/kev/refresh

Triggers KEV refresh (ADMIN/SUPERADMIN).

{
  "vulnsRead": 1234,
  "upserted": 1234
}

Feeds

GET /api/feeds

Lists configured public feeds (KEV, EPSS, NVD).

[
  {
    "type": "NVD",
    "enabled": true,
    "feedUrl": "https://services.nvd.nist.gov/rest/json/cves/2.0",
    "lastRefreshedAt": "2026-03-03T11:22:33+01:00",
    "lastStatus": "WARN",
    "lastMessage": "PARTIAL (skipped=18) processed=120/1234 (read=201, upserted=201)"
  }
]

POST /api/feeds/{feedType}/refresh

Triggers refresh for one feed (ADMIN/SUPERADMIN).

POST /api/feeds/refresh-enabled

Triggers refresh for all feeds in serial order: KEV -> EPSS -> NVD (ADMIN/SUPERADMIN).

POST /api/feeds/{feedType}/refresh-async

Starts one feed refresh in background and returns job metadata (ADMIN/SUPERADMIN).

POST /api/feeds/refresh-enabled-async

Starts all-feed serial refresh in background and returns created jobs (ADMIN/SUPERADMIN).

GET /api/feeds/jobs

Lists background feed refresh jobs (ADMIN/SUPERADMIN).

[
  {
    "jobId": "3f6b2a9d-7f04-4b0e-9176-78f3fd4b9c43",
    "feedType": "NVD",
    "status": "RUNNING",
    "createdAt": "2026-03-03T11:22:33+01:00",
    "startedAt": "2026-03-03T11:22:34+01:00",
    "finishedAt": null,
    "result": null,
    "message": "refresh in progress"
  }
]

POST /api/feeds/jobs/{jobId}/cancel

Requests cancellation of a background refresh job (ADMIN/SUPERADMIN).

PUT /api/feeds/{feedType}

Updates one feed settings (ADMIN/SUPERADMIN).

{
  "enabled": true,
  "feedUrl": "https://api.first.org/data/v1/epss"
}

Risks

GET /api/risks/top?limit=20

Returns highest-risk correlated rows (limit 1..200).

[
  {
    "hostname": "PC-001",
    "cveId": "CVE-2021-1234",
    "score": 812,
    "epssScore": 0.93,
    "advisoryCount": 4,
    "patchAvailable": true,
    "advisoryUrl": "https://vendor.example/advisory"
  }
]

Assets

GET /api/assets/search

Searches asset rows with optional filters (USER/ADMIN/SUPERADMIN).

Query params: q, hostname, product, vendor, version, cveId, minScore, maxScore, limit (default 200).

[
  {
    "hostname": "PC-001",
    "product": "Google Chrome",
    "vendor": "Google",
    "version": "122.0.6261.95",
    "cveId": "CVE-2024-12345",
    "score": 812,
    "epssScore": 0.93,
    "advisoryCount": 4,
    "patchAvailable": true
  }
]

GET /api/assets/hostnames?prefix=&limit=10

Returns top hostname suggestions for autocomplete (USER/ADMIN/SUPERADMIN).

GET /api/assets/products?prefix=&limit=10

Returns top software/product suggestions for autocomplete (USER/ADMIN/SUPERADMIN).

Error handling

Errors follow standard HTTP status codes.

{
  "status": 400,
  "error": "Bad Request",
  "message": "Missing CSV file"
}
